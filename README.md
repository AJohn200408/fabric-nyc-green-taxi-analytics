# NYC Green Taxi Analytics Platform

This project demonstrates an end-to-end Data Engineering solution built using Microsoft Fabric. The project processes NYC Green Taxi trip data through a Medallion Architecture (Bronze, Silver, Gold) using Microsoft Fabric Lakehouse, Notebooks, PySpark, and Pandas.

The solution includes data ingestion, transformation, validation, and modeling to create analytics-ready datasets. A Semantic Model and Power BI dashboard were developed to provide business insights, KPI monitoring, and interactive reporting.

## Technologies Used

* Microsoft Fabric
* Lakehouse
* PySpark
* Pandas
* SQL
* Semantic Models
* Power BI
* DAX

## Architecture

NYC Green Taxi Data → Bronze Layer → Silver Layer → Gold Layer → Semantic Model → Power BI Dashboard
